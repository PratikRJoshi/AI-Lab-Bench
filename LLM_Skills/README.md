# LLM Skills

AI/LLM skills and agent instructions for Cursor and similar AI coding assistants.

## Skills

| Skill | Description |
|-------|-------------|
| [url-truth-analyzer](./url-truth-analyzer/) | Transcribes video/audio URLs and performs truth-claim analysis. For medical content: EBM SORT analysis with peer-reviewed citations. For general science: claim validation with credible video sources. |

## Usage

These skills are designed for use with Cursor's `.cursor/skills/` directory. Copy the skill folder into your skills directory and reference it in your agent configuration.
